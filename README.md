## Supported Compilers

The following table outlines the supported compilers. Other
compilers may function correctly, but are not supported
as well as the compilers shown in the table below.

| Compiler Name | Minimum Version |
|---------------|-----------------|
| Clang         | 3.9             |
| GCC           | 4.9             |
| ICC           | 2018 update 2   |
